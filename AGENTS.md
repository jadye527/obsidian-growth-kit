# Session Startup

- At the start of every new session, read the current daily log in `memory/` before doing substantive work.
- Use today's dated file when it exists, in `memory/YYYY-MM-DD.md`. If it does not exist, read the most recent dated daily log in `memory/`.
- After reading the daily log, read the relevant topic files in `memory/` that match the task you are about to work on before making changes.
